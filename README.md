## 🏗️ What We Build

**Infrastructure & Platform**
- **[hive](https://github.com/christmas-island/hive-local)** — Cross-agent memory & task coordination (MCP-powered)
- **[crabitat](https://github.com/christmas-island/crabitat)** — K8s-based OpenClaw fleet deployments
- **[xmas-runner](https://github.com/christmas-island/xmas-runner)** — Purpose-built GitHub Actions self-hosted runner
- **[kubechecks](https://github.com/christmas-island/kubechecks)** — Validate Kubernetes changes before they hit the cluster

**Apps & Services**
- **[only-claws.net](https://only-claws.net)** — Agent registry & dashboard (where the claws live online)
- **[sclu](https://github.com/christmas-island/sclu)** — Standard Coors Light Units. How many Coors Lights is your drink worth?
- **[finagle](https://github.com/christmas-island/finagle)** — High-performance Go storage engine
- **[allegedly](https://github.com/christmas-island/allegedly)** — Semantic graph library backed by CockroachDB
- **[keyquest](https://github.com/christmas-island/keyquest)** — An adventure in typing mastery

**DevOps & Tooling**
- **[tf-modules-github](https://github.com/christmas-island/tf-modules-github)** + **[tf-live](https://github.com/christmas-island/tf-live)** — Terraform all the things
- **[go-scaffold](https://github.com/christmas-island/go-scaffold)** — Go project template with Dockerfile, k8s, CI
- **[stalwart](https://github.com/christmas-island/stalwart)** — Email infrastructure for only-claws.net
